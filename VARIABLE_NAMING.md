# 변수명 작성 가이드

## 📝 변수명 규칙

Robot Framework에서는 **영어뿐만 아니라 한글 변수명도 사용할 수 있습니다!**

## ✅ 사용 가능한 변수명

### 1. 영어 변수명 (일반적)

```robot
*** Variables ***
${MARKETING_POPUP_CLOSE}    22000000-0000-0000-F505-000000000000
${BROWSE_BUTTON}            37000000-0000-0000-F605-000000000000
${chart_play_all_button}    6C000000-0000-0000-F605-000000000000
```

### 2. 한글 변수명 (가능)

```robot
*** Variables ***
${마케팅_팝업_닫기}    22000000-0000-0000-F505-000000000000
${둘러보기_버튼}        37000000-0000-0000-F605-000000000000
${차트_전체듣기_버튼}    6C000000-0000-0000-F605-000000000000
```

### 3. 영어와 한글 혼용 (가능)

```robot
*** Variables ***
${MARKETING_POPUP_CLOSE}    22000000-0000-0000-F505-000000000000
${둘러보기_버튼}            37000000-0000-0000-F605-000000000000
${chart_button}             6C000000-0000-0000-F605-000000000000
```

## 💻 사용 예제

### 영어 변수명 사용

```robot
*** Test Cases ***
테스트 케이스
    Wait For Element And Click    id=${MARKETING_POPUP_CLOSE}
    Wait For Element And Click    id=${BROWSE_BUTTON}
```

### 한글 변수명 사용

```robot
*** Test Cases ***

    Wait For Element And Click    id=${마케팅_팝업_닫기}
    Wait For Element And Click    id=${둘러보기_버튼}
```

### 혼용 사용

```robot
*** Test Cases ***

    Wait For Element And Click    id=${MARKETING_POPUP_CLOSE}
    Wait For Element And Click    id=${둘러보기_버튼}
```

## 📋 변수명 규칙

### 필수 규칙

1. **변수는 `${변수명}` 형식으로 사용**
   ```robot
   ${변수명}    값
   ```

2. **변수명은 공백 없이 작성**
   ```robot
   # ✅ 올바름
   ${마케팅_팝업_닫기}
   ${MARKETING_POPUP_CLOSE}
   
   # ❌ 잘못됨
   ${마케팅 팝업 닫기}    # 공백 사용 불가
   ```

3. **언더스코어(_) 사용 가능**
   ```robot
   ${마케팅_팝업_닫기}    # ✅ 가능
   ${MARKETING_POPUP_CLOSE}    # ✅ 가능
   ```

### 권장 사항

1. **일관성 유지**: 한 파일 내에서 동일한 스타일 사용 권장
2. **의미 있는 이름**: 변수의 용도를 명확히 알 수 있는 이름 사용
3. **너무 긴 이름 피하기**: 가독성을 위해 적절한 길이 유지

## 🎯 선택 기준

### 영어 변수명 사용 시 (권장)

- ✅ 국제 표준 관례
- ✅ 다른 개발자와 협업 시 이해하기 쉬움
- ✅ 대부분의 예제와 문서가 영어 사용
- ✅ IDE 자동완성 지원이 더 좋을 수 있음

### 한글 변수명 사용 시

- ✅ 한국어 팀 내에서 이해하기 쉬움
- ✅ 변수 의미를 직관적으로 파악 가능
- ✅ 코드 가독성 향상 (한국어 사용자 기준)
- ⚠️ 일부 IDE나 도구에서 지원이 제한적일 수 있음

## 💡 실제 사용 예제

### 현재 프로젝트 스타일

```robot

# 영어 변수명 (현재 사용 중)
${MARKETING_POPUP_CLOSE}    22000000-0000-0000-F505-000000000000
${BROWSE_BUTTON}            37000000-0000-0000-F605-000000000000
```

### 한글 변수명으로 변경 예제

```robot
*** Variables ***
# 한글 변수명으로 변경 가능
${마케팅_팝업_닫기}    22000000-0000-0000-F505-000000000000
${둘러보기_버튼}        37000000-0000-0000-F605-000000000000
${차트_전체듣기_버튼}    6C000000-0000-0000-F605-000000000000
```

사용 예:
```robot
${팝업_닫힘}=    Wait For Popup And Close    ${마케팅_팝업_닫기}    wait_time=5    timeout=10
Wait For Element And Click    id=${둘러보기_버튼}
```

## ⚠️ 주의사항

1. **파일 인코딩**: UTF-8 인코딩 사용 권장
2. **공백 사용 불가**: 변수명에 공백 사용 불가 (언더스코어 사용)
3. **대소문자 구분**: Robot Framework는 대소문자를 구분합니다
4. **특수문자**: 일부 특수문자는 사용 불가 (언더스코어는 가능)
